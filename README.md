# DYMF Algorithm (DY-Microfinance Algorithm)

PyQt5 기반의 마이크로파이낸스 대출 계산 및 관리 시스템입니다. 다양한 대출 상품에 대한 이자 계산, 상환 일정 생성, 고객 신용 평가 등의 기능을 제공합니다.

## 프로젝트 개요

이 프로젝트는 DY마이크로파이낸스에서 소액 대출 업무를 효율적으로 관리하기 위해 개발한 데스크톱 애플리케이션입니다. 복잡한 대출 금리 계산, 상환 스케줄 생성, 고객별 대출 이력 관리를 자동화합니다.

**핵심 기능:**
- 원리금 균등상환 및 원금 균등상환 계산
- 유연한 대출 기간 및 이자율 설정
- 실시간 상환 계획 시뮬레이션
- CSV 파일 기반 대량 고객 관리

## 만든 알고리즘

### 1. 대출 이자 계산 알고리즘

#### 원리금 균등상환 (Equal Payment Loan / Amortized Loan)

**목표:** 매월 동일한 금액으로 원금과 이자를 함께 상환하는 방식

**월별 상환액 계산식:**
```
M = P × [r(1 + r)^n] / [(1 + r)^n - 1]
```

**변수:**
- M: 월별 상환액
- P: 대출 원금
- r: 월 이자율 (연 이자율 / 12)
- n: 총 상환 월수 (대출 기간 × 12)

**상환 일정 계산:**
- 매월 이자금 = 남은 원금 × 월 이자율
- 매월 원금 = 월별 상환액 - 월 이자금

#### 원금 균등상환 (Equal Principal Payment Loan)

**목표:** 매월 동일한 원금을 상환하고, 이자는 감소하는 방식

**월별 상환액 계산식:**
```
Mt = P/n + [P - P(t-1)/n] × r
```

**변수:**
- Mt: t번째 월의 상환액
- P: 대출 원금
- n: 총 상환 월수
- r: 월 이자율
- t: 현재 상환 월차

**특징:**
- 초기 상환액이 높고, 시간이 지나면서 감소
- 원리금 균등상환보다 총 이자금이 적음
- 고객이 초기 부담이 큼

## 기능

✅ **대출 상품 관리**
- 원리금 균등상환 상품
- 원금 균등상환 상품
- 커스텀 상환 방식 지원

✅ **대출 계산 및 시뮬레이션**
- 월별 상환액 자동 계산
- 상환 일정(상환표) 생성
- 시나리오별 상환 시뮬레이션

✅ **고객 관리**
- 고객 정보 등록 및 수정
- 대출 이력 관리
- CSV 파일 가져오기/내보내기

✅ **데이터 분석**
- 대출 통계 및 리포트
- 상환 현황 모니터링
- 연체 고객 추적

## 설치 및 실행

### 1. 사전 준비

- Python 3.10 이상 설치
- pip 최신 버전

### 2. 의존성 설치

```bash
pip install -r requirements.txt
```

### 3. 애플리케이션 실행

```bash
python -m app.main
```

또는

```bash
python main.py
```

### 4. Jupyter 노트북에서 알고리즘 검증

```bash
jupyter notebook
```

`algorithm/loan.ipynb` 파일을 열어 대출 계산 알고리즘을 상세히 확인할 수 있습니다.
